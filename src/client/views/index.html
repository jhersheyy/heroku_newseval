~About Sentiment Analysis & Natural Language Processing~

Sentiment Analysis

Sentiment Analysis uses Natural Language Processing (see below) to find the overall feeling in a block of text, such as a blog post, tweet, yelp review,etc. It's commonly used to get feedback from customers or employees, to check product reviews, and to monitor brands' reputations. Sentiment analysis is also known as opinion mining or emotion AI. A basic example is to determine the overall polarity of a text, whether it is more positive or negative, or if it's neutral.

While there are various methods to perform sentiment analysis, most of them involve breaking the text to be analyzed into smaller chunks like sentences or phrases. These smaller chunks are then analyzed and scored consistently, so the overall text is just the scores of the smaller chunks combined. For example, if a text has 3 phrases, and 2 are positive, but 1 is negative, then the text will be positive overall.

Here's a more general example looking at whole sentences instead of the parts making up a sentence:

  1. I really like their new album.
  2. I'm not sure if I like their new album.
  3. Their new album is terrible.
This example portrays a few aspects that sentiment analysis takes into account when processing texts. Although sentences 1 & 2 have similar words, sentence 2 uses negation with "not sure" whereas sentence 1 uses a simple statement "I like their new album" but with a modifier to exaggerate the positive feeling. Sentence 3 is also a simple statement, but in the negative direction. Taking this into account, sentence 1 will be classified as positive, and sentence 3 will be negative. On the other hand, sentence 2 would be classified as neutral since there's no real positive or negative emotion in it.

Natural Language Processing

Natural Language Processing (NLP) is a field of machine learning that focuses on analyzing human speech, or processing natural (human) language. This app uses Sentiment Analysis, but that is just one example of NLP. Other examples of NLP include transcribing text or categorizing and labeling text or speech. NLP typically achieves its goals by processing its input, human language, with patterns and phrases. Once an NLP model is taught one rule (a pattern), it solidifies its learning as it goes through more and more examples that use that pattern.

Sentiment analysis uses NLP and text mining to sort through human language. Text mining is also known as text analysis, because it gathers data on language and then implements NLP to find and share meanings and insights from the data. Text mining without NLP would require humans to look through all the data it collected, and using NLP removes the need for humans. Text mining and NLP work together to efficiently look through text.

Meaningcloud

https://www.meaningcloud.com

MeaningCloud is a Software as a Service product that specializes in text analysis. MeaningCloud offers various API's for developers to use. We used the API for sentiment analysis, but they also have API's for text classification, language identification, and summariziation (these are simply what they sound like, just so you know =] ) as well as many others.

https://www.meaningcloud.com/developer/sentiment-analysis MeaningCloud's Sentiment Analysis API can be used on a chunk of text, like the news articles used for this app, and even on the individual sentences making up the text. The API accepts a string of text, a url, or a selected document or file with analysis-friendly contents as valid queries. However, this app only supports the use of url's.

In the API used for main function for this app, the developers included various measures in the results. The most basic is polarity, but the API also analyzes how ironic the text is, gives it a confidence score, determines if the sentiments agree overall, and if the text given is objective or subjective.

Here are the analysis terms we use with their possible results: score_tag, agreement, subjectivity, irony, confidence
  • score: overall polarity (strong positive, positive, neutral, negative, strong negative, no sentiment)
  • agreement: whether polarities in each sentence/segment agree in polarity (agreement/disagreement)
  • subjectivity: whether overall text has any opinion or has fact only (subjective/objective)
  • irony: whether text contains any irony (ironic/nonironic)
  • confidence: level of confidence regarding analysis performed (0-100)
